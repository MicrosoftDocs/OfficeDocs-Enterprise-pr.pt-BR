# <a name="move-in-ready"></a>Move-in pronto

 **Resumo:** Obtenha os detalhes sobre o conjunto de nuvem opções de armazenamento que integram as ofertas de nuvem da Microsoft.
  
Soluções de armazenamento "Move-pol ready":
  
- Usar soluções de pré-empacotados imediatamente
    
- Está pronto para ir, incluído com serviços existentes
    
- Fornecer personalização limitada
    
As seções a seguir descrevem os detalhes de cada armazenamento de "movimentação de pronto" opção.
  
## <a name="sharepoint-online-in-office-365"></a>SharePoint Online no Office 365

### <a name="features"></a>Recursos

- Arquivos são acessíveis segura e confiável de vários dispositivos e de qualquer lugar
    
- Conteúdo é facilmente detectável por meio de pesquisa
    
- Fluxos de trabalho e políticas de retenção facilitam a usar e manter o conteúdo
    
- Conteúdo pode ser acessado por meio de APIs da web ou um navegador, ou offline
    
### <a name="common-uses"></a>Usos comuns

- Portal de empresa ou organização
    
- Colaboração baseado em equipe
    
- Pesquisa corporativa
    
### <a name="key-storage-scenarios"></a>Cenários de armazenamento de chave

- Colaborar
    
- Comunicações da empresa
    
- Gerenciar vídeos
    
### <a name="resources"></a>Recursos

Para obter informações adicionais, clique [aqui](https://support.office.com/article/SharePoint-Online-and-OneDrive-for-Business-software-boundaries-and-limits-8f34ff47-b749-408b-abc0-b605e1f6d498).
  
Para obter informações de custo, clique [aqui](https://products.office.com/business/compare-office-365-for-business-plans).

## <a name="onedrive-for-business"></a>OneDrive for Business

### <a name="features"></a>Recursos

- O armazenamento é seguro, confiável e acessível a partir de vários dispositivos e de qualquer lugar
    
- Conteúdo é facilmente detectável por meio de pesquisa pelo usuário
    
- Conteúdo pode ser acessado por meio de APIs da web ou um navegador, ou offline
    
- Cada usuário obtém 1 TB de armazenamento
    
### <a name="common-uses"></a>Usos comuns

- Armazenamento de documentos de usuários individuais
    
- Colaboração ad hoc em documentos
    
### <a name="key-storage-scenarios"></a>Cenários de armazenamento de chave

- Arquivos de backup

## <a name="azure-backup"></a>Backup do Windows Azure

### <a name="features"></a>Recursos

- Protege perfeitamente dados recebidos do Windows Server e Windows Server Essentials
    
- Inclui o System Center Data Protection Manager
    
- Os dados são armazenados no armazenamento geo-replicado, quais mantém seis cópias de seus dados em dois datacenters Azure
    
- Suporta as alterações delta, frequência definida, compactação, criptografia, maior retenção e limitação de largura de banda
    
### <a name="common-uses"></a>Usos comuns

- Proteger os dados de uma variedade de fontes de Windows Azure (VMs) e local (SharePoint, SQL, Exchange)
    
### <a name="key-storage-scenarios"></a>Cenários de armazenamento de chave

- Arquivos de backup
