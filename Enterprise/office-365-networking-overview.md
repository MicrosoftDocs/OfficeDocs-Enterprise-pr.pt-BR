<span data-ttu-id="f46e3-p111">O Office 365 foi projetado para um desempenho ideal usando conexões de rede criptografada, reduzindo a necessidade de imposição de segurança de perímetro e segurança do ponto de extremidade. Since user data and processing is distributed between many Microsoft datacenters, there is no single network endpoint to which client machines can connect. <span data-ttu-id="f46e3-144">Certos problemas comuns de desempenho são criados quando o tráfego do Office 365 está sujeito a inspeção de pacotes e de saída centralizada:</span><span class="sxs-lookup"><span data-stu-id="f46e3-144">Certain common performance issues are created when Office 365 traffic is subject to packet inspection and centralized egress:</span></span>
