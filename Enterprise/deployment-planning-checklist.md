# <a name="deployment-planning-checklist-for-office-365"></a>Lista de verificação de planejamento de implantação do Office 365

Quando você move uma organização empresarial para o Office 365, é importante planejar exatamente quais etapas que você deseja obter, quando executá-las, e que o executará. Esta lista de verificação ajudarão sua organização ao planejar e preparar para uma migração para o Office 365. As fases e etapas na lista de verificação são alinhadas com as diretrizes fornecidas pelo [Centro de inclusão](https://go.microsoft.com/fwlink/?LinkId=517115). Se sentir livres para adaptar-se esta lista de verificação para as necessidades da sua organização.

## <a name="need-help-with-your-deployment"></a>Precisa de ajuda com sua implantação?
Você deseja ajuda para obter o Office 365 configurar? Considere usar [FastTrack](https://fasttrack.microsoft.com/office) ou a [consultores de implantação para serviços do Office 365](deployment-advisors-for-office-365.md).
